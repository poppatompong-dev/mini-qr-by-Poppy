# Product

## Register

product

## Users
Citizens, municipal officers, computer technicians, and developers under Nakhon Sawan Municipality who need to generate custom styled QR codes (single/batch) and share multiple files via a secure QR code.

## Product Purpose
MiniQR is a lightweight, high-performance, and elegant client-side QR code generator. It allows custom styling (dots, corners, logos, frames) and batch generation, and enables multi-file zipping and uploading to a Supabase bucket to create dynamic file-download QR codes.

## Brand Personality
Modern-municipal, trustworthy, clean, premium, and highly functional. It should feel like a premium tool, avoiding clunky government dashboard tropes.

## Anti-references
- Clunky web interfaces with basic flat forms.
- Generic un-styled QR codes.
- Hidden features or menus that make files upload difficult to find.
- Bland, low-contrast gray text on off-white backgrounds.

## Design Principles
- **Immediate Discovery**: Core utilities like URL encoding and file uploads must be accessible in one click from the dashboard.
- **Refined Premium Contrast**: Rich color theme transitions (dark/light), premium glassmorphic touches (used selectively), and high-contrast readable typography.
- **Clear Guidance**: Informative messages in Thai if uploads fail due to database bucket setup or file size limits.

## Accessibility & Inclusion
- Contrast ratios meeting WCAG AA standards.
- Full responsive support across all devices (mobile, tablet, desktop).
- Dark mode preference set natively or toggled via menu.
